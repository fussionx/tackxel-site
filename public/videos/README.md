# Video testimonials

Drop client testimonial MP4s here. The homepage "Hear it from our clients"
section and its lightbox player are already wired to these filenames:

- `testimonial-1.mp4`
- `testimonial-2.mp4`
- `testimonial-3.mp4`

To change filenames, client names, roles, or quotes, edit `videoTestimonials`
in `src/app/page.tsx`. Recommended: H.264 MP4, 16:9, ≤ ~20 MB each for fast load.
